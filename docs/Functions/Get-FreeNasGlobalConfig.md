# Get-FreeNasGlobalConfig

## SYNOPSIS
This function return the global configuration for your FreeNas Server

## SYNTAX

```
Get-FreeNasGlobalConfig [<CommonParameters>]
```

## DESCRIPTION
This function return the global configuration for your FreeNas Server

## EXAMPLES

### EXEMPLE 1
```
Get-FreeNasGlobalConfig
```
